What is ax + b = cx + d

ax + b = cx + d is an equation in which a, b, c, d and x are all numbers. When it says to solve the eqation, you need to work out the value of x. a, b, c, and d are constants, whereas x is a variable.

How to solve ax + b = cx + d

  1. First of all, you need minus b from both sides, so now the equation is ax = cx + d - b
  2. Next, you need to minus cx from both sides, so now the equation is ax - cx = d - b
  3. Then, you need to divide both sides by (a - c), so now the equation is x = (d - b) / a
  4. Now do the minusing and dividing and you should have your answer!

A step-by-step example

The example I'll give is 3x + 2 = x + 8

  1. Minus the 2 from both sides giving 3x = x + 6
  2. Minus the x from both sides giving 3x - x = 6
  3. Simplify that to 2x = 6
  4. Divide 6 by 2 which gives you 3.
  5. So 3 is your answer!
