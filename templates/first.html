What is ax + b = c

ax + b = c is an equation in which a, b, c and x are all numbers. When it says to solve the eqation, you need to work out the value of x. a, b, and c are constants, whereas x is a variable.

How to solve ax + b = c

  1. First of all, you need minus b from both sides, so now the equation is ax = c - b
  2. Then, you need to divide both sides by a, so now the equation is x = (c - b) / a
  3. Now do the minusing and dividing and you should have your answer!

A step-by-step example

The example I'll give is 2.5x + 3 = 8

  1. Minus the 3 from both sides giving 2.5x = 5
  2. Divide 5 by 2.5 which gives 2.
  3. So 2 is your answer!
